(Table with parameter, decription (data type, and notes columns))
-------------------------------------------------------------------
CampaignId: (required)
Value specific to your Program (${viewId}). (Integer)

ActionTrackerId: (required)
Value specific to each conversion event  (${eventId}). (Integer)

EventDate: (required)
Submit one of the following formats:
ISO-8601 format
Example: 2017-08-01T23:55:24-08:00
dd-MMM-yyyy HH:mm:ss z
Example: 01-AUG-2017 23:55:24 PST

IpAddress: (recommended)
Public Internet IP Address of the customer. The IP address is used to help the system identify fraudulent activity. (String 128)

ClickId: (required)
Send the Click Id value captured on the landing page URL. (String 128) 

OrderId: (required)
Unique trackable ID for each event. (AlphaNumeric 64)

CustomerId: (recommended)
Unique customer identifier your platform assigns to customer accounts. Do not use personally identifiable data for this field. Pass an empty string if unavailable. (String 255)

CustomerEmail: (recommended)
SHA1 hash of customer’s email. Pass an empty string if unavailable. (String 64)

CustomerStatus: (recommended)
Use values of “New” or “Returning”. This value can be used to alter payouts. (AlphaNumeric 40)

OrderPromoCode: (recommended)
Pass a single promo code (stacked promo codes are not supported).  Pass an empty string if unavailable. This value can be used to alter payouts. (String 64)
