PAGE LOAD API

Make a POST request to the Page Load API each time a user loads a page.

API Endpoint
POST https://trkapi.impact.com/PageLoad

API Call Example
POST /PageLoad HTTP/1.1
Host: trkapi.impact.com
Accept: application/json
Content-Type: application/json
Content-Length: 111

{
  "CampaignId": ${viewId},
  "PageUrl": "pageUrl",
  "ReferringUrl": "refUrl",
  "CustomProfileId": "cpid-123",
  "CustomerId": 657584,
  "CustomerEmail": "3be1c5898e7d600b2765f964e27cf0af531c4970",
  "EventDate": "dd-MMM-yyyy HH:mm:ss z",
  "UserAgent": "useragent",
  "IpAddress": "ipaddress"
}
