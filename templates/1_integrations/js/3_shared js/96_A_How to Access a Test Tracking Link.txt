How to Access a Test Tracking Link
    From a test partner account
        Log in to your test partner account
        Using the "Create A Link" widget, make sure your desired brand is selected
            In most cases, you should only see one brand available in the widget drop down menu
        If desired, input a landing page
            This is only necessary if you need to test a page other than the default landing page
        Click the "Create" button to generate the test tracking link
    From your impact.com Brand account
        Log in to your impact.com brand account
        Navigate to Engage > Content > Ads
        Expand the Actions drop down menu for any ad and select "Send To Media"
        On the Send Ads screen, select your test partner account in the Partner field
        The Ad Code field will update with the script traditionally used to embed the ad
